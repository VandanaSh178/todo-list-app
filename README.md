# 📝 To-Do List App

A sleek, responsive, and feature-rich to-do list web app built with HTML, CSS, and JavaScript. Organize your daily tasks with built-in features like timers, priorities, filters, and theme switching. All data is saved locally so your tasks persist even after refreshing.

## 🚀 Features

- ✅ Add, complete, and delete tasks
- ⏱ Per-task countdown timer (Pomodoro-style: 25 minutes default)
- 🟩 Live progress bar and completion stats
- 🌗 Toggle between dark and light themes (auto-saved)
- 🎯 Filter tasks by status: All, Completed, Pending
- 🔺 Assign priority to tasks (High, Medium, Low)
- 💾 Data persistence using `localStorage`
- 🎉 Confetti animation when all tasks are completed

## 🖥️ Tech Stack

- HTML5
- CSS3 (Flexbox, Dark/Light Themes)
- JavaScript (ES6+, DOM manipulation, timers)
- localStorage (for saving data)
